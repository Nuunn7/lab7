## Build хийх

### Ant ашиглан:
```bash
ant compile          # Compile хийх
ant javadoc          # Documentation үүсгэх
ant all              # Бүгдийг хийх
ant clean            # Цэвэрлэх
```

### IDE дээр:
- Eclipse/IntelliJ IDEA дээр "Run as JUnit Test"
